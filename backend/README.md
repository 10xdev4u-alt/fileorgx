# 🧠 Smart File Organizer - Backend

Python backend for Smart File Organizer Pro.

## 🚀 Setup

1.  **Install Poetry**:
    ```bash
    curl -sSL https://install.python-poetry.org | python3 -
    ```

2.  **Install dependencies**:
    ```bash
    poetry install
    ```

3.  **Run with Uvicorn**:
    ```bash
    poetry run uvicorn api.server:app --reload
    ```
